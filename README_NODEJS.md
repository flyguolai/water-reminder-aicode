# 定时喝水提醒工具 - Node.js版本

这是一个使用Node.js实现的定时提醒工具，每15分钟会弹出一个通知，提醒用户活动身体并喝水。

## 功能特点

- 每15分钟自动发送桌面通知
- 可自定义提醒消息
- 支持声音提醒
- 简单易用，无需复杂配置

## 安装依赖

```bash
npm install
```

## 使用方法

```bash
npm start
```

## 依赖说明

- node-notifier: 用于发送桌面通知
- node-schedule: 用于实现定时任务

## 注意事项

- 程序需要保持运行状态才能持续发送提醒
- 请确保您的系统允许显示桌面通知
- Node.js版本要求: 12.x 或更高版本
